Grouped by access level. React SPA serves /, /trading/*, /picks/*. Static HTML for /workbench/*.

5. Weekly Content Rhythm

Publishing cadence — pipeline writes Firestore data, React picks pages display it dynamically.

Monday AM
Weekly Picks
  • Analyse stocks in Workbench
  • Build strategies in Strategy Builder
  • Publish tiles to Firebase
  • Run: npm run picks:create
  • /picks page auto-updates (Firestore)
  • Share link on social + newsletter
Wednesday
Mid-Week Update
  • Check positions — any hit targets?
  • Close tiles that reached profit target
  • Adjust or roll positions if needed
  • Active tiles auto-refresh via pipeline
Friday PM
Weekly Review
  • Run: npm run picks:close
  • Script closes remaining open tiles
  • Calculates actual P&L per position
  • /picks/recap page auto-updates (Firestore)
  • Cron job updates pick_outcomes daily
